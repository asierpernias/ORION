qué puede hacer ORION?

[ VOZ ]

Comandos por voz

Transcripción local con Whisper. Sin internet, sin latencia.

[ LLM ]

IA local con Ollama

Corre modelos como phi3 o llama3 directamente en tu máquina.

[ CMD ]

Comandos del sistema

Abre apps, busca archivos, lanza scripts con lenguaje natural.

[ NOTE ]

Notas rápidas

Dicta o escribe notas. Se guardan localmente en texto plano.

[ TIME ]

Timers y recordatorios

Pon timers y alertas por voz. Todo sin salir del teclado.

[ PRIV ]

100% privado

Cero datos enviados a la nube. Todo ocurre en tu hardware.

cómo instalarlo

01

Descarga el paquete

Elige tu sistema operativo arriba y descarga el zip correspondiente.

02

Instala dependencias

Dentro de la carpeta extraída, ejecuta:

pip install -r requirements.txt
03

Instala Ollama y descarga un modelo

ORION necesita Ollama corriendo localmente. Descárgalo en ollama.com y luego:

ollama pull phi3:latest
04

Ejecuta ORION

Abre el ejecutable y descubre todas sus posibilidades

05

Activa el micrófono (opcional)

Para comandos por voz, asegúrate de tener un micrófono activo. Whisper medium se descarga automáticamente en el primer uso (~1.5GB).
