> features

qué puedes hacer ORION?

[ VOZ ]

Comandos por voz

Transcripción local con Whisper. Sin servidores externos.

[ TEXT ]

Barra de texto

Abre con Espacio. Respuesta sin latencia de red.

[ SEARCH ]

Búsqueda inteligente

Google y YouTube con detección automática de motor.

[ NOTES ]

Notas locales

Guarda notas con timestamp. Sin cloud, sin cuenta.

[ TIMER ]

Timers

Múltiples timers simultáneos. Beep al terminar.

[ APPS ]

Abrir apps

Busca apps en Windows, macOS y Linux dinámicamente.

> setup

cómo instalarlo

01

Descarga el .zip

Descarga ORION_Setup_1.0.0.zip, extrae y ejecuta el installer.

02

Instala Ollama (opcional)

Para IA local, instala Ollama y ejecuta:

ollama pull phi3:latest
03

Lanza ORION

Doble clic para hablar, Espacio para escribir.

04

Desde código fuente

git clone https://github.com/asierpernias/ORION
pip install -r requirements.txt
python app.py
